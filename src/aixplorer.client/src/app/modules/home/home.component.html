To start streaming,

  • go to Settings , select camera and microphone, click "Save",
  • display the User List , click the name of the (logged-in) user you want to connect to,
  • click "Connect". Both will receive a message with the streaming link.
For the best experience, we recommend using one of the following browsers on mobile phones, as they have been tested: Chrome, Firefox, or Brave.
Additionally, we encourage you to consult the Technical Notes for more information.
